Follow these steps to run the app directly on your iPhone and see live changes during development.

1. Update the file fridge9000/mobile/src/services/config.ts and change to your PC IP

2. Run this command it from source directory: docker compose up --build.make sure the build has ended.

3. Prerequisites (One-time setup)
Download Expo Go: Search for and install the "Expo Go" app from the Apple App Store.

Network Check: Ensure your iPhone is connected to the same Wi-Fi network as the computer running the project.

4. How to Connect
open the Expo app and enter the URL: exp://10.100.102.5:8081 manually.
